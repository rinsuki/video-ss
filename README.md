# video-ss

タブのスクショを撮ってvideoタグのところを切り抜くやつ。
主にZenzaWatch向け

## 使い方

1. git cloneする
1. yarn installする
1. yarn buildする
1. manifest.jsonがあるところを chrome://extensions → パッケージ化されてない拡張機能を読み込む(ないときは右上のデベロッパーモードをONにする) から読み込む
1. 適当に動画を開く
1. 右上のVを押す
1. スクショが撮れる
1. うれしい
